# Nexora Studio — Complete Website Brief for Claude AI

## 1) Role and Objective
You are a senior brand strategist, UX designer, copywriter, and front-end web developer.

Your task is to design and generate a **premium, modern, conversion-focused IT services website** for a company called **Nexora Studio**.

The site should feel:
- modern
- premium
- trustworthy
- intelligent
- scalable
- polished
- enterprise-ready
- startup-friendly

This is **not** a generic agency site. It should position Nexora Studio as a **strategic technology partner** that helps businesses modernize operations, build digital products, and improve efficiency through software, cloud, AI, automation, and managed IT services.

## 8) Visual Direction
Create a visual system that feels like a modern, premium technology brand.

### Overall Style
- sleek
- minimal
- elegant
- futuristic but not flashy
- highly readable
- spacious layout
- premium dark-mode leaning design

### Recommended Theme
Use a **dark modern tech aesthetic** with subtle gradients, glassmorphism accents, soft shadows, refined borders, and strong typography.

### Suggested Color Palette
Use colors close to this:
- Background: `#0B1020`
- Surface: `#121A2B`
- Primary Accent: `#4F8CFF`
- Secondary Accent: `#7B61FF`
- Highlight / Cyan Accent: `#35D6FF`
- Text Primary: `#F5F7FB`
- Text Secondary: `#A8B3C7`
- Border: `rgba(255,255,255,0.08)`

### Typography
Use a premium modern pairing such as:
- Headings: **Sora** or **Manrope**
- Body: **Inter**

### UI Feeling
- rounded corners (large but elegant)
- high-end cards
- subtle hover states
- soft gradients
- animated section reveals
- refined call-to-action buttons
- strong contrast for accessibility
